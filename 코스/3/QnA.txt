반복문 중에서 for문을 쓰는 규칙
for(인덱스 변수 = 처음 값; 인덱스 변수의 마지막 값; 인덱스 변수++){

}
여기서 인덱스 변수를 미리 선언해도 되고, 아니면 for문장 속에서 선언해도 된다.
int i;
for(i=0; i<10; i++){
  // 10번 반복하기
}

아니면,
for(int i=0; i<10; i++){
  //10번 반복하기
}

두 방법의 차이는? 실행값은 같음
그러나, 1번은 미리 i를 선언했기에 for 문장이 끝난 뒤 i를 다른 용도로 사용할 수 있다.
2번은 for문 안에서 i를 선언했기에 for문 밖에서 i를 사용할 수 없다.
왜냐면? 어떤 변수라도 선언한 위치에 속한 블록(구역) 안에서만 사용이 가능하기 때문이다.
